# Student Checklist

Use this checklist while you publish your existing website.

- [ ] I created a GitHub repo
- [ ] I uploaded the contents of my project folder
- [ ] I can see `index.html` at the root of the repo
- [ ] I turned on GitHub Pages
- [ ] I copied my live Pages URL
- [ ] I cloned the repo in Antigravity
- [ ] I made one small edit
- [ ] I committed
- [ ] I pushed
- [ ] I checked the live site
- [ ] I submitted both links
